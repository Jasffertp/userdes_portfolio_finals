A1: MY FAVORITE APPLICATION

MY FAVORITE APPLICATION

One of the apps that I really enjoy using is Lithium. Lithium is a epub/ebook reading app that allows the user to read any epub that they’ve download, I’m a huge fan of reading books and during this quarantine this app has been helping me to still keep reading books. I first started using this app in 2017, back then I was visiting different websites and downloading pdf’s so that I can continue books that I’ve been reading, I also used to buy physical book but every time I would go out, I wouldn’t be able to bring any of my books and every time I would read a pdf I would forget what was the last page I read. With lithium I was able to read a lot of books on the go, it’s amazing because It felt like I had a library in my pocket and with the internet it felt like I could read any books that I want. The thing that made me really love this app was its features, unlike the other reading apps once you open Lithium you’re immediately directed to last page of the last book that you just read and it has this small feature where you can change the theme of the app, you can make it dark, white or just a light color that easy for the eyes, you can also change it to other color but it’s for premium users only, even though I’m using the free version of this app it’s still very cool. Another feature that I really like is how you turn to the next page, instead of scrolling down you can swipe or tap the next page, it’s a small feature but I really like it.
