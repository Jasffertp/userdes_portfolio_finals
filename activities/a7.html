A7: PERSONAL WEBSITE REPORT

TYPOGRAPHY

For the interactive parts of my website (Buttons, links etc.) I used ‘Open sans pro’


For the overall of my website I used ‘Source sans pro’ for headers and ‘roboto’ for the remaining parts


I changed the size for the text if it is a sub header, it the image above the text ‘G1: USER DESIGN’ is the main header of the page, and that part is what I want the user to immediately see therefore I made it bigger and the text below ‘1. WHAT IS DESIGN’ is the sub header that is also using source sans pro but is smaller but not smaller than the text below, which is using roboto font, to catch the users attention after seeing the main header.

COLOR PALETTE

I used the following as the color palette of my website, snow for the color of the background of my website, Jet for the fonts and the navbar, and raisin black when the mouse hovers over the texts of the navbar

GLOBAL NAVIGATION

To the left of my website is a fixed side bar that is in every part of the website


When you hover over the texts a new sidebar will appear showing the activities, reflection, guide questions done
